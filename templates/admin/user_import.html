Formát CSV souboru

Sloupce (oddělené čárkou nebo středníkem):

username, email, first_name, last_name, role
  • username — povinný, unikátní přihlašovací jméno
  • email — volitelný
  • first_name — volitelný
  • last_name — volitelný
  • role — volitelný: user (výchozí), admin, owner

Hlavička je nepovinná. Kódování: UTF-8.
Existující uživatelé budou aktualizováni (email, jméno). Noví dostanou náhodné heslo.
