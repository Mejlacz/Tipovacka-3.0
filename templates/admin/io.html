👥 Týmy a uživatelé

Import ze souboru. Podporované formáty: CSV (UTF-8 s nebo bez BOM) a XLSX (Excel). Staré .xls není podporováno — ulož jako .xlsx nebo .csv.

👥 Import týmů (CSV)

Sloupce: name, sport, display_name, alias, category. Existující týmy (stejný name+sport) se aktualizují.

📊 Import týmů (XLSX)
