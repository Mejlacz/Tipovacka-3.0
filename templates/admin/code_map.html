Jádro aplikace

  • RequireAdmin / RequireLogin / RequireApproved — auth middleware

leaderboard.go

  • Leaderboard — hlavní žebříček soutěže
  • LeaderboardChartData — data pro graf vývoje bodů
  • MyRankAPI — aktuální pořadí přihlášeného (AJAX)
