← Admin

Kdy co může selhat a kde to hledat. Vždy začni Audit logem — chyby jsou tam zapsané včetně přesného DB erroru.

{{/* ─── Zápasy a výsledky ─────────────────────────────────────────── */}}

⚽ Zápasy a výsledky

❌Zápas nejde přidat

  • Týmy nejsou přiřazeny k soutěži → přejdi na Správa týmů soutěže
  • home_team_id nebo away_team_id = 0 (form neposlal ID)
  • DB INSERT selhal — přesný důvod je v audit logu (action: match_create)
  • Kontrola duplikátů — zápas stejných týmů ve stejný den už existuje
admin_matches.go — AdminMatchCreate admin_teams.go — AdminCompetitionTeamsSave admin.go — AdminCompetitionSetDeadline

❌Výsledek nejde zadat / přepsat

  • DB UPDATE selhal — zkontroluj audit log (action: match_score)
  • Zápas je already finished a formulář to nekontroluje
  • Přepočet bodů selhal (chyba v RecalculateTips) → logy v konzoli
admin_matches.go — AdminMatchSetResult common.go — RecalculateTips common.go — RecalculateStandings

❌Body se nepřepočítaly po zadání výsledku

  • RecalculateTips neprošla — chyba v SQL, špatné competition_id
  • RecalculateStandings selhal — chyba v standings tabulce
  • Hromadné výsledky (BulkResults) — každý zápas počítá zvlášť, jeden neúspěch ostatní neblokuje
common.go — RecalculateTips, RecalculateStandings admin_matches.go — AdminBulkResultsSubmit admin_matches.go — AdminMatchSetResult

❌Zápas nejde smazat

  • Na zápas existují tipy → je potřeba nejdřív smazat tipy uživatelů (AdminRemoveUserTips), nebo DB cascade
  • DB DELETE selhal — audit log (action: match_delete)
admin_matches.go — AdminMatchDelete admin_matches.go — AdminRemoveUserTips

❌Datum zápasu nejde nastavit / je špatně zobrazeno

  • Parsování datetime-local formátu selhal (očekává se "2006-01-02T15:04")
  • Časové pásmo — server vs Prague — viz /admin/time
  • TZ override aktivní — zkontroluj diagnostiku času
admin_matches.go — AdminMatchSetDate, AdminMatchEdit common.go — NowPrague, SetTZOffsetOverride admin_time.go — AdminTimeDiag
{{/* ─── Tipy a deadline ────────────────────────────────────────────── */}}

🎯 Tipy a deadline

❌Tip nejde uložit

  • Deadline soutěže nebo zápasu vypršel → tipování blokováno
  • Uživatel není schválen (is_approved = false)
  • Zápas je already finished (is_finished = true)
  • DB UPSERT selhal — audit log (action: tip_save)
tips.go — SubmitTip, SubmitTipAjax common.go — IsBeforeDeadlineComp admin.go — AdminCompetitionSetDeadline

❌Deadline nefunguje správně (tipy jdou/nejdou když nemají)

  • Deadline je NULL → tipování otevřeno do začátku každého zápasu
  • Soutěžní deadline vs. datum zápasu — bere se minimum
  • TZ problém — deadline je v UTC, zobrazuje se v Prague čase
  • Prázdný deadline_str při nastavení → deadline se smaže
common.go — IsBeforeDeadlineComp admin.go — AdminCompetitionSetDeadline admin_matches.go — AdminMatchesList (zobrazení deadline)

❌Admin přepsal tip uživatele a šlo to vrátit

  • Audit log ukládá old_value při admin_set_tip → přejdi na Audit log › Tipy
  • Undo funguje pro posledních 100 undoable akcí
admin_matches.go — AdminSetTip admin_audit.go — AdminAuditUndo
{{/* ─── Uživatelé a přístupy ───────────────────────────────────────── */}}

👤 Uživatelé a přístupy

❌Přihlášení nefunguje

  • Špatné heslo nebo email — audit log (action: login_fail)
  • Uživatel je zablokován (is_blocked = true)
  • Uživatel čeká na schválení (is_approved = false) → přesměruje na /pending
  • Sloupec email neexistuje v DB → přihlášení jen přes username
auth.go — LoginSubmit common.go — GetCurrentUser, RequireLogin common.go — InitUserSchema (sloupce)

❌Reset hesla nefunguje (zapomenuté heslo)

  • Email se neodešle → viz sekce Email níže
  • Token expiroval — tokeny mají platnost 1 hodinu
  • APP_URL není nastavena ve Fly.io secrets → odkaz v emailu směřuje špatně
auth.go — ForgotPasswordSubmit, ResetPasswordSubmit admin_email.go — sendEmail

❌Admin práva nefungují / uživatel nevidí admin sekci

  • is_admin = false v DB → nastav přes AdminUserSetRole nebo toggle
  • Sloupec is_admin neexistuje → InitUserSchema vrátí false pro userCols.IsAdmin
  • Session expirovala → uživatel musí se znovu přihlásit
  • Audit log (action: user_toggle_admin, user_role)
common.go — RequireAdmin, InitUserSchema admin.go — AdminUserToggleAdmin, AdminUserSetRole

❌Uživatel nejde schválit / zablokovat

  • DB UPDATE selhal — audit log (action: user_approve, user_block)
  • Sloupec is_approved / is_blocked neexistuje → InitUserSchema
  • Pokus o zablokování owner účtu bez owner oprávnění
admin.go — AdminUserApprove, AdminUserBlock, AdminUserUnblock common.go — InitUserSchema

❌Hromadný import uživatelů selhal

  • Duplicitní username nebo email → INSERT selže pro daný řádek, ostatní pokračují
  • Špatný formát CSV/textu → parsování vrátí prázdný slice
admin_user_import.go — AdminUserImportSubmit admin.go — buildUserInsertSQL
{{/* ─── Import dat ─────────────────────────────────────────────────── */}}

📥 Import dat

❌API import fotbalu nefunguje (football-data.org)

  • API klíč expiroval nebo není nastavený → Fly.io secret FD_API_KEY
  • Rate limit — bezplatný plán má omezení na volání za minutu
  • Tým z API nenalezen v DB → resolveOrUpsertTeam vytvoří nový
  • fd_code soutěže chybí → nastav přes editaci soutěže
admin_api_import.go — AdminAPIImport, AutoFetchResults admin_api_import.go — resolveOrUpsertTeam admin.go — AdminCompetitionEditSubmit (fd_code)

❌API import hokeje nefunguje (thesportsdb.com)

  • League ID nebo season string špatně zadán
  • API vrací jiný formát → cleanHockeyTeamName může selhat
  • Timeout na vzdálenou API
admin_api_hockey.go — ashImport, ashUpdateResults admin_api_hockey.go — cleanHockeyTeamName, tsdbCall

❌OCR import / textový import nefunguje

  • Parsování nerozpoznalo formát textu → zkontroluj parseMILine / parseOCRText
  • Tým z textu nenalezen fuzzy matchingem → přiřadí se jako nový nebo prázdný
  • Session expirovala mezi Parse a Confirm → data ztracena, je potřeba znovu uploadovat
  • Žádná soutěž nebyla vybrána → compID = 0
admin_match_import.go — parseMILine, AdminMatchImportConfirm admin_ocr.go — parseOCRText, AdminOCRConfirm admin_match_import.go — matchTeamsWithKnown

❌Import / export CSV nebo XLSX selhal

  • Špatný formát souboru nebo kódování (UTF-8 vs CP1250)
  • Chybí závislost excelize → build selže
  • Export prázdný — žádná data v DB pro danou soutěž
admin_export.go — AdminGeneralExportCSV, AdminGeneralExportXLSX admin_teams.go — AdminTeamsImportCSV, AdminTeamsImportXLSX admin_export.go — AdminTipsImportSubmit

📧 Email a notifikace

❌Email se neodešle

  • SMTP přihlašovací údaje chybí → Fly.io secrets: SMTP_HOST, SMTP_USER, SMTP_PASS
  • Port 587 / TLS blokován Fly.io sítí
  • Prázdný email uživatele → send_welcome odmítne
  • Audit log (action: user_send_welcome) zachytí chybu
admin_email.go — sendEmail, sendEmailHTML admin_email.go — AdminEmailSend admin.go — AdminUserSendWelcome auth.go — ForgotPasswordSubmit

❌Notifikace o zápase se neodeslaly

  • Uživatel nemá povoleny notifikace (notification_settings tabulka)
  • sendEmail selhal → viz Email výše
  • Zápas byl přidán bez data → notifikace se nespustí bez match_date
notify.go — sendMatchNotificationForID admin_matches.go — AdminMatchNotifyNow profile.go — ProfileNotifications

❌Push notifikace nefungují

  • VAPID klíče nejsou nastaveny ve Fly.io secrets
  • Uživatel odvolal povolení v prohlížeči
  • Subscription expirovala → je potřeba se znovu přihlásit k push
profile.go — ProfilePushSubscribe, ProfilePushTest notify.go — notifySendEmail

🏟️ Týmy

❌Tým nejde přidat k soutěži / chybí v selectu

  • Tým není v competition_teams tabulce → přiřaď ho přes Správa týmů soutěže
  • Tým má jiný sport než soutěž → filter v SQL
admin_teams.go — AdminCompetitionTeamsSave admin_matches.go — AdminMatchesList (načítá týmy pro select)

❌Sloučení týmů selhalo

  • Zdrojový a cílový tým jsou stejné → odmítnuto
  • FK závislosti (tipy, zápasy) — merge přesouvá, ne maže
  • Audit log (action: team_merge)
admin_teams.go — AdminTeamMerge, AdminTeamMergeBulk admin_user_merge.go — AdminUserMerge (analogické pro uživatele)

🎁 Bonusové otázky

❌Automatické vyhodnocení nedalo body

  • Správná odpověď není nastavena → nastav ji přes Uložit správnou odpověď
  • Fuzzy matching nenašel shodu — levenshtein vzdálenost nad prahem
  • Odpovědi jsou sloučeny → unmerge a znovu evaluate
admin_extra.go — AdminExtraAutoEvaluate, isFuzzyMatch admin_extra.go — AdminExtraSaveCorrect, AdminExtraMergeAnswers

❌Odpověď na bonusovou otázku nejde uložit

  • Otázka je uzavřena (is_closed = true)
  • Deadline bonusových otázek vypršel
  • Audit log (action: extra_save)
extra.go — ExtraSaveAjax admin_extra.go — AdminExtraQuestionToggleClose, AdminExtraDeadlineSettings
{{/* ─── Infrastruktura a deploy ────────────────────────────────────── */}}

⚙️ Infrastruktura a deploy

❌Deploy selhal (GitHub Actions)

  • Build error v Go → zkontroluj Actions log na GitHubu
  • FLY_API_TOKEN expiroval → obnov v GitHub Secrets
  • Fly.io nedosažitelný → status.fly.io
  • Stav: curl https://api.github.com/repos/Mejlacz/Tipovacka-3.0/actions/runs?per_page=3
.github/workflows/fly-deploy.yml main.go — lifespan, init

❌Server nereaguje / timeout

  • Fly.io machine zaspal (free tier) → keepalive /admin/keepalive
  • DB connection pool vyčerpán → zkontroluj Health dashboard
  • Paměťový limit překročen → Fly.io dashboard
admin_keepalive.go — AdminKeepaliveEnable admin_health.go — AdminHealthDashboard database/ — get_db, Pool

❌Neon záloha (sync) nefunguje

  • NEON_BACKUP_URL není nastavena ve Fly.io secrets
  • Zdrojová a cílová DB mají jiné schéma → copyTableNeon selže na sloupci
  • Sync tabulky (neon_sync_config, neon_sync_log) neexistují → InitNeonSyncTables()
admin_neon_sync.go — RunNeonSync, copyTableNeon admin_neon_sync.go — InitNeonSyncTables

❌Čas / datum se zobrazuje špatně

  • Server běží v UTC, Prague time se počítá přes pragueLocation
  • TZ override aktivní → resetuj na /admin/time
  • Naive vs aware datetime při porovnání → přidej ::timestamptz v SQL
admin_time.go — AdminTimeDiag, LoadTZOverrideFromDB common.go — NowPrague, pragueLocation
