Co dělá Keep Alive?

GitHub Actions workflow keepalive.yml pinguje endpoint /health každých 10 minut. Díky tomu Koyeb neuspí aplikaci při nečinnosti. Pokud workflow vypneš, aplikace může přejít do spánku a první request trvá déle.
